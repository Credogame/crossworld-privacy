Политика конфиденциальности — Crossworld

Эффективность с: 26 июня 2025 года

Владелец: Credo

1. Общая информация

Crossworld является бесплатной игрой с монетизацией через рекламу. Мы не собираем персональные данные напрямую, однако используем Google AdMob SDK, который автоматически собирает определённые данные, как описано ниже.

2. Какие данные собирает AdMob SDK

Google Mobile Ads SDK может собирать и передавать следующие данные:

  • Идентификаторы устройства – Android Advertising ID, App Set ID и/или другие ID устройства (рекламный ID).
  • IP‑адрес – для определения приблизительного местоположения пользователя.
  • Диагностические данные – аварии приложения, время запуска, метрики взаимодействия (нажатия/просмотры).
  • Рекламные данные – рекламный контент, метрики показа, кликов, впечатлений.

3. Цели использования данных

Собранные данные используются для:

  • Персонализации рекламы
  • Анализа эффективности рекламных кампаний
  • Улучшения работы SDK и предотвращения мошенничества

4. Передача данных третьим лицам

Собранная информация передаётся:

  • Компании Google и её рекламным партнёрам для показа таргетированной рекламы
  • Возможно, другим сторонним рекламным и аналитическим сетям через платформу AdMob

5. Защита данных

Все данные передаются по защищённому соединению (TLS/HTTPS) и обрабатываются в соответствии с политиками безопасности Google.

6. Управление идентификатором рекламы

Пользователь имеет право отключить рекламный идентификатор в настройках Android (Google settings → Ads) или сбросить его. В Crossworld можно сделать это непосредственно в настройках устройства.

7. Согласие на персонализированную рекламу (GDPR, ЕС)

Для пользователей из ЕС перед показом персонализированной рекламы мы используем Google UMP SDK, чтобы получить явное согласие. Без выраженного согласия показ рекламы будет неконкретизированным или по общим настройкам Android.

8. Права пользователей

Пользователь вправе:

  • Отключить рекламный ID или сбросить его
  • Отказаться от персонализированной рекламы (на уровне Android или Google Settings)
  • Связаться с разработчиком для удаления/запроса информации
  • Изменить своё согласие в европейских странах
  • Ознакомиться с политикой Google: How Google uses data

9. Обновления политики

Мы оставляем за собой право обновлять политику (например, при добавлении новых SDK или сборе данных). Дата последнего обновления будет указана вверху. Рекомендуем отслеживать изменения.
